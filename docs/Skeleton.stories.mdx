# Skeleton

骨架屏占位在某些情境下相对于菊花图来说，用户体验要好很多，它能减少用户等待时的挫败感，提升用户体验。
`Skeleton`默认渲染一个段落占位。

<Preview withSource="open">
    <Story name="默认">
        <Skeleton />
    </Story>
</Preview>

`Skeleton`组件也提供了一些预设的占位形式
