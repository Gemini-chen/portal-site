# Picture

`Picture` 组件不是原生的`picture`元素，而是`image`元素,但提供了`object-fit`和`object-position`的 pylyfill，可以直接使用这些属性。当所有的资源均加载失败时，会默认显示 fallback。

---

### basic example

    <Story name="standard">

### fallback

<Preview withSource="open">
    <Story name="fallback">
        <Picture
            style={{
                width: 200,
                height: 200
            }}
            source="http://localhost/404.jpg"
        />
    </Story>
</Preview>

如果不需要 fallback,将 fallback 设置为 `null`，图片在加载失败后会自动隐藏(display)。
