`portal-site` 是公司用于开发cms前端部分的一系列组件库，包括提供路由上下文和数据请求的 `portal-core`，提供数据结构处理的`portal-data-cms`,提供常用的功能组件的`portal-elements`

### ✨ Feutures
- 使用Typescript开发，提供完整的类型定义
- 内置声明式的 redux api 框架 🎈[mpa](https://github.com/wyyufeng/mpa)
- 提供增强的 `React-Router` 组件
- 提供清晰的路由表
- 提供常用的功能组件和增强的原生元素
- 使用 `css-in-js` 提供了基础的样式，方便修改
